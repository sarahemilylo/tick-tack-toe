## 🎮 Features

- 2-player local gameplay (Player 1 = X, Player 2 = O)
- Click-based grid interaction
- Automatic turn switching
- Win detection (rows, columns, diagonals)
- Tie detection
- Score tracking for both players
- Click anywhere to reset after a game ends

---

## 🕹️ How to Play

1. Player 1 (X) starts first  
2. Click on any square in the grid to place your mark  
3. Players alternate turns  
4. First player to get 3 in a row wins  
5. If all squares are filled with no winner → tie  
6. After a game ends, click anywhere to start a new round
